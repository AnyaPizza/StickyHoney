## Craft Sticky Pistons with a Bottle of Honey instead of Slimeballs!

This datapack adds the option to use a Bottle of Honey instead of just Slimeballs if you so choose.

Based on the datapack [Slimy Honey!](https://modrinth.com/datapack/oszoukauas-slimy-honey!) by oSzoukaua. If you want any versions below 1.21.11, check there.
